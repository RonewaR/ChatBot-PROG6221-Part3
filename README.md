 Cybersecurity Awareness Chatbot

 Overview
A C# Windows Forms chatbot designed to promote cybersecurity awareness through interactive conversations, quizzes, task management, voice features, and personalized responses.

 Features
- GUI-based chatbot interface
- Voice output 
- Cybersecurity tips and awareness information
- Activity log tracking
- Task Assistant (add, view, complete, and clear tasks)
- Cybersecurity quiz with scoring
- Sentiment detection
- Basic NLP/keyword recognition

 Technologies Used
- C#
- Windows Forms
- .NET Framework
- SpeechSynthesizer
- Lists and Dictionaries

 Example Commands
- `hello`
- `time`
- `date`
- `activity`
- `tip`
- `start quiz`
- `add task change password`
- `show tasks`
- `complete task 1`

 Project Files
- **Program.cs** – Application entry point
- **Form1.cs** – Chatbot logic and features
- **Form1.Designer.cs** – GUI design
- **VoiceGreeting.cs** – Voice functionality
